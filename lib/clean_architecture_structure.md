# Clean Architecture Scaffold

This project has been prepared with a clean architecture folder structure. No implementation code is included yet, only placeholder files to preserve the structure.

## Structure

- `lib/core/`
  - Shared core components, use cases, utilities, error handling, and network helpers.
- `lib/features/feature_example/`
  - A sample feature module broken into `data`, `domain`, and `presentation` layers.
- `lib/shared/`
  - Shared widgets, themes, and utilities used across features.

## Notes

- Replace `feature_example` with your actual feature names.
- Add actual Dart files under these folders as feature requirements become clear.
